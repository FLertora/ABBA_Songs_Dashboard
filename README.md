# ABBA_Songs_Dashboard

Dashboard interactivo desarrollado en Power BI para analizar la evolución de las canciones de ABBA a lo largo del tiempo.
Permite visualizar tendencias, distribución por décadas y métricas clave de forma clara e intuitiva.

Key Insights
- Evolución de canciones por año
- Distribución de canciones por década
- Identificación de periodos con mayor producción musical
- Análisis temporal mediante segmentaciones (slicers)

Tools & Technologies
- Power BI
- DAX (medidas básicas)
- Power Query
- Excel (fuente de datos)
